# Local Analysis Dashboard

A Streamlit-based dashboard for AI safety research analysis and methodology testing.

## Features

- **Methodology Testing**: Test detection methods on AI responses
- **Result Visualization**: Charts and graphs for research analysis  
- **Data Export**: Export results for external analysis
- **Reproducible Research**: Consistent analysis environment

## Usage

```bash
# Launch local dashboard
python src/main.py dashboard

# Access at http://localhost:8501
```

## Components

- Real-time methodology testing
- Research result aggregation and analysis
- Trend visualization across studies
- Export capabilities for academic papers
